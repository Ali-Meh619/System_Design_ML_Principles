# 30-Minute System Design Round

## Time Box

- 0-5 min: requirements and constraints
- 5-10 min: estimates
- 10-18 min: high-level design
- 18-26 min: one deep dive
- 26-30 min: failures, metrics, trade-offs

## Required Outputs

- one architecture diagram
- one bottleneck analysis
- one rollback/failure path
- one scaling plan for 10x growth
