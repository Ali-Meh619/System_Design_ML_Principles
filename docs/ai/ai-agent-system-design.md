# AI Agent System Design

## Agent Anatomy

- planner
- tool executor
- memory/context manager
- policy/safety guardrails
- evaluator

## Multi-Agent Patterns

- orchestrator-worker
- planner-critic
- specialist swarm with aggregator

## Safety Controls

- tool permission boundaries
- prompt injection defense
- output validation and policy checks
- human approval gates for high-impact actions

## Reliability Concerns

- deterministic retries for tool calls
- bounded execution steps
- traceability per decision
